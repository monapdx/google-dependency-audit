# Contributing

Thanks for contributing!

## Development setup

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Pull requests

- Keep changes focused and small.
- If you adjust scoring logic, include a brief note explaining why.
- Prefer readable, explicit code over clever code.

## Style

- Use clear variable names (this app is meant to be legible).
- Keep the questionnaire copy empathetic and non-judgmental.
